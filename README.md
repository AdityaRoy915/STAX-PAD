# Stack-Pad  
A custom 5-key hackpad designed with a clean, compact layout and precise cutouts for switches, keycaps, and controller components. This design is made for creating a personalized macro pad that fits your workflow and style.

## About  
Stack-Pad is a minimal 5-key macro pad built for productivity, shortcuts, and customization. Whether you're editing videos, coding, or gaming, this compact device lets you assign custom macros to each key for faster and more efficient control.

## How to Use :
- Insert the switches into the custom 3D printed case.  
- Mount the PCB inside the case and align it properly.  
- Solder the switches and connect them to the PCB.  
- Attach the microcontroller (if separate) and connect via USB.  
- Place keycaps on the switches.  
- Upload your firmware (like QMK/VIA) and start using your macros.  

## Features :
- 5 programmable mechanical keys  
- Compact and ergonomic design  
- Fully customizable layout and macros  
- Supports common firmware (QMK/VIA)  
- DIY-friendly assembly
